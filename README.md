# Nethmini Vinodya — Portfolio

A nature-inspired personal portfolio built with plain HTML, CSS, and JavaScript.

## Features

- Hero, About, Skills, Education, Projects, Achievements, and Contact sections
- Liquid glass navigation, cards, and buttons
- Falling leaf animations
- Animated skill bars and scroll reveals
- Fully responsive layout
- Accessibility and reduced-motion support

## Getting Started

Open `index.html` in your browser, or use a local server:

```bash
npx serve .
```

## Customize

Edit content directly in `index.html`. Update images in the `Assets/` folder and social links in the Contact and Footer sections.
